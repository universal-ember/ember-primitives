# Get Started

With `ember-primitives` you only ship as much JS to your users as you need. Import whatever you want. 

## Install 

```bash 
pnpm add ember-primitives 
```

## Compatibility

- ember-source 4.8+
- Glint 1.0+
- TypeScript 4.8+
- ember-auto-import 2.6+
- embroider 3.0+
